# Planning Document
Below is the rough planning of the building `Eat What?`.

### Timeframe
The projected timeline for this project is about 6 weeks as of 22/12/2025. Out of the 6 weeks, 4 weeks will be "truncated". I.e. only available after office hours. While the other 2 weeks will be full days.

### Project Phases
The project will be divided into 5 phases. 

#### Phase ~~1~~ 2 - Creating the Backend Template (7 days)
Creating a backend template for our backend services. Implementing/integrating with supabase for authentication as well as database.

End goal: Reusable backend template that can be reused in different projects.

#### Phase ~~2~~ 1 - Learning React Native (10 days)
Immersing in tutorial hell for React Native. RN is something new so will take some time to learn how to build with this tech and understanding the ecosystem

End goal: Understand the basics of React Native and able to build features.


#### Phase 3 - Creating the React Native Template (3 days)
After tutorial hell, to build a simple RN template for easy scaffolding of new mobile app. 

End goal: Reusable RN template that can be reused in different projects.


#### Phase 4 - Building the core features (15 days)
Build the main features of `Eat What?`. This phase is where the majority of the building will be.

End goal: A viable product ready for deployment.

#### Phase 5 - Deployment (2 days)
Last but not least, deployment of the final product. To understand how mobile apps are published and updated.

End goal: `Eat What?` on the app store

#### Buffer
A 5 days buffer is set aside
